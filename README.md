# netlist-tools

Набор CLI-утилит для работы со списками доменов (`*.domains`), поддоменов (`*.subdomains`) и IP-адресов (`*.ipname`).
Полученные списки можно использовать для раздельной маршрутизации, блокировки или фильтрации трафика.

## Возможности
- Обновление выбранных категорий (каскадный DNS-резолв: `*.domains` -> `*.subdomains` -> `*.ipname`)
- Агрегация и экспорт выбранных категорий в единые списки
- Автоматическая сортировка, удаление дубликатов и оптимизация диапазонов

## Утилиты и структура проекта
| Элемент | Тип | Назначение |
|:---|:---|:---|
| `Lists/` | Каталог | Исходные списки, разбитые по категориям. Обновляются утилитой `lists.exe`. |
| `Lists.Aggregated/` | Каталог | Списки, получаемые в результате агрегации выбранных категорий утилитой `lists.exe`. |
| `Sources/` | Каталог | Исходный код утилит на Python. |
| `lists.exe` | Утилита | Менеджер списков. Объединяет выбранные категории доменов, поддоменов и IP-адресов в общие списки. Списки IP-адресов агрегируются с заданным CIDR-префиксом (по умолчанию: /32). Также обновляет списки поддоменов и IP-адресов для доменов выбранных категорий. |
| `ip2rsc.exe` | Утилита | Конвертер для MikroTik. Преобразует списки IP-адресов в адрес-листы RouterOS. |
| `ipaggr.exe` | Утилита | Агрегатор IPv4. Объединяет смежные адреса в оптимальные CIDR-блоки, сокращая размер списков. |
| `ipgrab.exe` | Утилита | Извлекатель IPv4. Парсит логи, конфиги или бинарные файлы, извлекая все найденные IP-адреса. |
| `ipname.exe` | Утилита | Резолвер. Преобразует IPv4-адреса, сети и хосты в формат `IP-адрес # Имя_хоста`. |
| `sdaggr.exe` | Утилита | Агрегатор поддоменов. Группирует, сортирует и оптимизирует списки, удаляя дубли. |
| `sdfind.exe` | Утилита | Искатель поддоменов. Находит активные поддомены для заданных основных доменов. |

## Пример рабочего процесса
1. Поместите исходные списки доменов в папку `Lists/` с расширением `*.domains`.
2. Запустите `lists.exe`.
3. Выберите нужные категории.
4. Выберите `Update` для обновления. Списки поддоменов и IP-адресов обновятся и сохранятся в `Lists/` с исходными именами.
5. Выберите `Aggregate` для объединения. Результат сохранится в `Lists.Aggregated/`.
6. При необходимости сконвертируйте полученный список IP-адресов в адрес-лист MikroTik с помощью `ip2rsc.exe`.

## Установка
1. Распакуйте архив в любую директорию.
2. Запускайте утилиты из `Far`, `cmd`, `PowerShell` или другой оболочки.
3. Для вывода справки используйте ключ `-h`.
